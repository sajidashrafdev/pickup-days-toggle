# Pickup Days Toggle

A simple WordPress plugin to toggle pickup days and control Elementor tabs visibility.

## Features

- Enable/Disable pickup days (Monday–Sunday)
- Works with Elementor tabs using IDs (tab-monday, tab-tuesday, etc.)
- Lightweight and easy to use
- No ACF required

## Requirements

- WordPress
- WooCommerce
- Elementor (for tab UI)

## Installation

1. Upload the plugin to `/wp-content/plugins/`
2. Activate from WordPress dashboard
3. Go to **Pickup Days** menu
4. Enable/Disable days
